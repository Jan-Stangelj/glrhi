glrhi (Open\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+GL\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+R\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}endering \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+H\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}ardware \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+I\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}nterface) is a library aimed at abstracting Modern Open\+GL 4.\+6 using modern concepts, like DSA (Direct state acces) to reduce boilerplate at no preformance overhead.\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Building}{Building}}\label{index_autotoc_md3}
\hypertarget{index_autotoc_md4}{}\doxysubsection{\texorpdfstring{Linux}{Linux}}\label{index_autotoc_md4}
Requirements\+:
\begin{DoxyItemize}
\item CMake 3.\+16
\item A C++ 20 compiler
\item GNU Make
\item A GPU supporting Open\+GL 4.\+6 (not required for building, but for using the library)
\end{DoxyItemize}

Build process\+:

First clone the repo. \begin{DoxyVerb}    git clone --recursive https://github.com/Jan-Stangelj/glrhi.git
\end{DoxyVerb}


Then make the build directory and cd to it. \begin{DoxyVerb}    cd glrhi
    mkdir build
    cd build
\end{DoxyVerb}


Configure the project \begin{DoxyVerb}    cmake ..
\end{DoxyVerb}


And compile it (The j option sets the amount of threads make will use) \begin{DoxyVerb}    make -j 4
\end{DoxyVerb}

To use the library in a CMake project, first add it as a subdirectory. \begin{DoxyVerb}    set("GLRHI_BUILD_EXAMPLES" OFF CACHE BOOL "") # Disables building examples
    add_subdirectory("vendor/glrhi")
\end{DoxyVerb}


Finally link the library to your executable. \begin{DoxyVerb}    target_link_libraries("your_executable" glrhi)
\end{DoxyVerb}
